- [同步 iOS/ipadOS 提醒到其日历的脚本](./ios-reminder-sync.js)

  在苹果的提醒和日历 APP 中建立相同名称的提醒列表和日历本，执该脚本将把该名称的提醒列表同步到日历本中。可添加到快捷指令中自动执行。
